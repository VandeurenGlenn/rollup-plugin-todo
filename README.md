# rollup-plugin-todo [![NPM version][npm-image]][npm-url] [![Dependency Status][daviddm-image]][daviddm-url]
>

## Installation

```sh
$ npm install --save rollup-plugin-todo
```

## Usage

```js
const rollupPluginTodo = require('rollup-plugin-todo');

{
  ...
  plugins: [
    rollupPluginTodo()
  ],
  ...
}
```
